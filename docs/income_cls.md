
    
##### 简要描述

- 经济能力10W-v9

##### 请求URL
- ` http://server:port/yrzx/finan/net/10w/v9 `(server:port在联调时提供)
  
##### 对接方式
- 请求方式：http/get
- 编码:UTF-8
- 输出格式：JSON
- 注：account与key由商户提供

##### 输入参数

| 参数名称 | 是否必填 | 是否签名 | 参数描述 |
|:--- |:--- |:--- |:---  |
| account | 是 | 是 | 账户 |
| idCard | 是 | 是 | 身份证号(最后一位X大写) |
| name | 否 | 是 | 姓名 |
| mobile | 是 | 是 | 手机号 |
| reqid | 是 | 是 | 请求流水号,不大于20位 |
| verify | 是 | 否 | 签名Md5(account+idCard+mobile+reqid+key).toUpperCase |

##### 返回参数

|字段名称|是否签名|描述|
|:-----  |:-----|-----                           |
|code |是   |返回码，具体见返回码字典  |
|msg |否   |返回码的中文描述，具体见返回码字典  |
|uid |是   |流水号  |
|reqid |否   |请求流水号，不大于20位  |
|result |否   |二级节点  |
|verify |否   |签名  |

###### result二级节点

| 字段名称 | 描述 |
|:--- |:---  |
| range | 收入模型评分 |

##### 返回码字典

| 返回码 | 描述 |
|:--- |:---  |
| 001 | 成功 |
| 999 | 查无结果 |
| 002 | 账号不存在 |
| 003 | 余额不足 |
| 004 | 请给该账户授权 |
| 005 | 身份证号为空或格式错误 |
| 006 | 姓名格式错误 |
| 020 | 参数为空或格式错误 |
| 008 | reqid为空或长度大于20 |
| 009 | 账号为空 |
| 011 | verify为空或格式错误 |
| 012 | 接口错误，请联系提供商 |
| 013 | 校验签名错误 |
